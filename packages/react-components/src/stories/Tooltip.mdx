### Accessibility

An active tooltip can be dismissed by pressing the <kbd>Escape</kbd> key, in addition to a blur or mouse-out event.

Learn more about the [ARIA tooltip role at MDN](https://developer.mozilla.org/en-US/docs/Web/Accessibility/ARIA/Roles/tooltip_role).

## Configuration

### Tooltip sits inside a TooltipTrigger

To make a tooltip appear, it must sit inside of a `TooltipTrigger`, as a sibling of an element that is hoverable or focusable.

```jsx
<TooltipTrigger>
  <Button>Hover or focus</Button>
  <Tooltip placement="top">
    This text appears when the button is hovered or focused.
  </Tooltip>
</TooltipTrigger>
```

### `children`

Use the `children` prop for the descriptive text within the tooltip.

### Placement

The optional `placement` prop determines where the tooltip displays in relation to the trigger element within `TooltipTrigger`.
