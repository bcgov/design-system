## Configuration

A radio group is made up of two components: `RadioGroup` (the parent container) and any number of `Radio` components (each representing a single option.)

To populate a `RadioGroup`, pass an array of `Radio` components to the `children` slot.

### Radio

Each `Radio` expects:

- A unique identifier, set via the `value` prop
- A text label, passed to the `children` slot

<Canvas of={RadioGroupStories.RadioGroupTemplate} />

To disable an individual item, use the `isDisabled` prop on a `Radio`:

### RadioGroup

Use the `label` and `description` props to explain the purpose of an input. If you do not use a visible `label`, you must instead use `aria-label` or `aria-labelledby` to label the input for assistive technologies.

Use `orientation` to toggle a radio group between `vertical` (default) and `horizontal` layouts:
