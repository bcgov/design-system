## Configuration

### Variants

The inline alert ships with 4 themes for different use cases. Each theme has a different colour scheme and icon.

The alert theme is set by the `variant` prop:

- `info`
- `success`
- `warning`
- `danger`

If no `variant` prop is passed, the alert defaults to the `info` theme.
