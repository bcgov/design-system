### Anatomy

An Accordion can be used singularly, or wrapped inside an AccordionGroup.

AccordionGroup expects some number of Accordions, passed as children:

```javascript
<AccordionGroup>
  <Accordion />
  <Accordion />
  <Accordion />
</AccordionGroup>
```
