# Design tokens

Design tokens are semantically-named variables that define granular design decisions like colour, typography and spacing.

Tokens are consumed internally by all components in the design system, to ensure a consistent visual language. You can also consume them directly to style other parts of your UI. Design tokens are published as both CSS and JavaScript variables.

To use design tokens, install the [@bcgov/design-tokens](https://www.npmjs.com/package/@bcgov/design-tokens) package via npm:

```bash
npm i @bcgov/design-tokens
```

To learn more about the tokens schema, [review the documentation](https://www2.gov.bc.ca/gov/content?id=5993A3D597F1454CB0E5DFA02DBB2EB7).
