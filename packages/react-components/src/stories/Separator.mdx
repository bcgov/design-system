## Usage and resources

A separator provides visual separation between UI elements. It renders as a solid line.

## Configuration

### Orientation

The `orientation` prop defaults to `horizontal`, and renders an `<hr>` element:

<Canvas of={SeparatorStories.HorizontalSeparator} />
<Controls of={SeparatorStories.HorizontalSeparator} />

Set `orientation` to `vertical` to render a `<div>` with the `separator` role:

<Canvas of={SeparatorStories.VerticalSeparator} />
<Controls of={SeparatorStories.VerticalSeparator} />
