# Security policy

## Supported versions

### `design-system-react-components`

| Version | Supported          |
| ------- | ------------------ |
| 0.x.x   | :white_check_mark: |

### `design-tokens`

| Version | Supported          |
| ------- | ------------------ |
| 3.x.x   | :white_check_mark: |
| < 3.0   | :x:                |

## Reporting a vulnerability

If you identify a potential vulnerability in either of the B.C. Design System's packages, you can:

- [Report it privately via GitHub](https://github.com/bcgov/design-system/security/advisories/new) (recommended)
- Email designsystem@gov.bc.ca

When reporting a vulnerability, please use [the GitHub Security Lab template](https://github.com/github/securitylab/blob/main/docs/report-template.md).
